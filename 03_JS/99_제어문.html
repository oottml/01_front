제어문

프로그램의 흐름을 제어하는 문법

* 조건문

조건에 따라 프로그램의 흐름을 제어하는 문법

  • if 문
  • switch 문
if문 
        if (조건식) {
            // 조건식이 참(true)인 경우 실행할 코드
        }

        * 조건식의 결과는 true 또는 false의 값을 가짐
        * 조건식에는 관계 연산자 또는 논리 연산자가 사용됨
          - 관계 연산자 : >, <, >=, <=, ==, !=, ===, !==
          - 논리 연산자 : &&, ||
if ~ else 문 
        if (조건식) {
            // 조건식이 참(true)인 경우 실행할 코드
        } else {
            // 조건식이 거짓(false)인 경우 실행할 코드
        }
if ~ else if ~ else 문 
        if (조건식1) {
            // 조건식1이 참(true)인 경우 실행할 코드
        } else if (조건식2) {
            // 조건식1이 거짓(false)이고, 조건식2가 참(true)인 경우 실행할 코드
        } else {
            // 조건식1, 조건식2가 모두 거짓(false)인 경우 실행할 코드
        }
switch 문 
        switch(비교대상) {
            case 값1:
                // 비교 대상과 값1이 동일한 경우 실행할 코드
                break;  // 조건문 종료
            case 값2:
                // 비교 대상과 값2가 동일한 경우 실행할 코드
                break; 
            default:
                // 비교대상의 값이 case 항목에 모두 해당되지 않는 경우 실행할 코드
                // => 마치 if문의 else와 같음
        }

* 반복문

동일한 작업을 반복하여 실행하고자 할 때 사용하는 문법

  • for 문
    • 기본 for문
    • for in 문
  • while 문
for 문 
        * 기본 for 문

        for(초기식;조건식;증감식) {
            // 조건식이 참(true)인 경우 실행할 코드
        }
        + 초기식 : 반복문의 흐름을 제어할 변수를 초기화
        + 조건식 : 반복문을 실행시킬 조건에 대한 식
        + 증감식 : 초기화한 변수의 값을 증감시키는 식

        => 프로그램의 흐름
          초기식 -> 조건식 -> 조건식이 참인 경우 -> 중괄호 내의 코드 실행 -> 증감식
                -> 조건식 -> 조건식이 참인 경우 -> 중괄호 내의 코드 실행 -> 증감식
                -> 조건식 -> 조건식이 거짓인 경우 -> 반복문 종료
    

        * for in 문

        for (let 변수명 of 배열명) {
            // 배열의 첫번째 위치부터 마지막 위치까지 반복할 코드 => 배열 길이만큼 반복
            // 배열의 각 인덱스 위치의 아이템에는 변수명을 통해 접근 가능!
        }
* while 문 
        [초기식]        // 조건식에서 사용될 변수에 대한 초기화
        while (조건식) {
            // 조건식이 참(true)인 경우 실행할 코드

            [증감식] // 초기화된 변수를 변경하기 위한 증감식
        }
* do ~ while 문 
        do {

            // 처음 1회는 조건식의 결과에 상관없이 실행
            // 그 이후는 조건식이 참(true)인 경우 반복적으로 실행할 코드
            
        } while(조건식);
