이미지 태그 : img

* src 속성 : 이미지 경로

* alt 속성 : 이미지 설명

- 이미지 경로가 잘못되었을 경우 대체할 텍스트
- 시각 장애인들을 위한 스크린 리더에서 이미지에 대한 설명을 읽어줄 수 있음

* width, height 속성

이미지의 가로 길이(width), 세로 길이(height) 길이 조정 가능
고정길이(px), 가변길이(%) 지정 => 기본값 : px

고정 길이(px) => 기본값(단위)

화면 크기가 변경되어도 이미지의 크기는 고정(변동 없음)

가변 길이(%)

화면 크기에 따라 이미지의 크기가 변경
부모 요소 크기에 따라서도 이미지의 크기가 변경
